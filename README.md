# DevCamper API

> Backend API for the dev camper application which is a bootcamp directory

## Usage

Rename the config/config.env.env to config.env and update the setting

## Install Dependencies

```
npm install

```

## Run app

```
# Run in dev enviroment
npm run dev
# Run in production
npm start

```
